# 基于腾讯wujie实现前端微应用

<br/>

[wujie传送门](https://wujie-micro.github.io/doc/)

<br/>

以wujie@1.0.17为基础

<br/>

## 各应用技术栈情况

- Portal： vue3+vite+router@4+pinia
  
- app-vue3: vue3+vite+router@4+pinia
  
- vue2 : vue2+ webpack+ router@3

<br/>

## 端口

- portal: 3100

- app-vue3: 3101

- vue2: 3102

<br/>
